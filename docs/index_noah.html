Mastermind Trivia

About this quiz

These questions will start easy and get harder–so watch out! Everyone will play against the mastermind on their given topic.

About the questions

Below are the types of questions you’ll see:

  1. Multiple-choice
    • The easiest question to get yourself situated
  2. Open
    • It gets a little harder when you don’t have options anymore
  3. Ranking
    • Can you put them in the right order?

About the questions, cont’d

  1. Sound-bite
    • I’ll play a sound clip and you’ll have to answer the question related to it
  2. Range
    • Unlike the other questions so far, whoever is closest will get this point
  3. Who/What am I?
    • You’ll get 3 clues, if you can get the person/object on the first clue that’ll be worth 3 points, 2 for the second clue, and 1 for the last clue–pending you get the right answer

About the scoring

We’ll calculate the differentials between the mastermind and the rest of the team–whoever has the highest differential wins.

However, if there is a tie I have some volleyball trivia questions to break it.
